# FASB Conceptual Framework

## Core Idea

The FASB Conceptual Framework is a coherent system of interrelated objectives and concepts that governs how financial accounting standards are developed. Think of it as the "constitution" behind U.S. GAAP: individual standards must be consistent with it, even though the framework itself is not an authoritative standard that overrides specific pronouncements.

The framework answers three layered questions:

1. **Why** does financial reporting exist? → To provide decision-useful information to capital providers.
2. **What** qualities make information useful? → Fundamental and enhancing qualitative characteristics.
3. **What** should be reported? → The elements of financial statements.

## Objectives of Financial Reporting

The primary objective is to provide information useful to existing and potential investors, lenders, and other creditors in making decisions about providing resources to the entity. This grounds every subsequent concept in user needs rather than preparer convenience.

## Qualitative Characteristics

```
FUNDAMENTAL                    ENHANCING
───────────────────────────    ──────────────────────────────
Relevance                      Comparability
  ├─ Predictive value          Verifiability
  └─ Confirmatory value        Timeliness
                               Understandability
Faithful Representation
  ├─ Completeness
  ├─ Neutrality
  └─ Freedom from error
```

**Relevance** asks: does the information make a difference to a decision? **Faithful representation** asks: does the depiction match the economic phenomenon it claims to describe? Both must be present for information to be genuinely useful. Enhancing characteristics improve usefulness but cannot rescue information that fails the fundamental tests.

A **cost constraint** overlays the entire framework: the benefit of reporting information must justify the cost of providing it.

## Elements of Financial Statements

The framework defines ten elements that populate financial statements:

| Balance Sheet Elements | Income / Equity Elements |
|---|---|
| Assets | Revenues |
| Liabilities | Expenses |
| Equity | Gains |
| | Losses |
| | Investments by owners |
| | Distributions to owners |
| | Comprehensive income |

Each element has a precise definition (e.g., assets are probable future economic benefits controlled by the entity as a result of past transactions) that standard-setters use when deciding whether a new item qualifies for recognition.

## Why a Deterministic Reference Matters

Because the framework is closed-form and definitional, exam questions test exact vocabulary. Confusing "relevance" with "verifiability," or misidentifying an element, produces a wrong answer regardless of reasoning quality. Memorizing the precise hierarchy and definitions is therefore non-negotiable.

## Prereqs

- Basic understanding of what financial statements are (balance sheet, income statement)
- Familiarity with the distinction between GAAP standards and conceptual guidance
- General knowledge of who uses financial reports (investors, creditors)
